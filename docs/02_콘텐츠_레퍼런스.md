## 8. 합성 시스템

### 합성 조건
- 같은 희귀도의 아이템 **10개** → 새 아이템 **1개**
- 상수: `SYNTHESIS_REQUIRED = 10` (`src/data/items.ts`)

### 합성 결과 확률

| 입력 희귀도 | 결과 희귀도 (확률) |
|------------|------------------|
| 1★ 일반 | 1★ 45% / **2★ 55%** (하락 없음) |
| 2★ 레어 | 1★ 5% / **2★ 45%** / **3★ 50%** |
| 3★ 전설 | 3★ 100% (더 높은 등급 없음) |

> **주의**: 1★ 합성 시 하락(-1) 페널티는 발생하지 않음 (코드: `if (rand < 5 && inputRarity > 1)`)

### 합성 후 처리
- 사용한 아이템 10개 인벤토리에서 제거
- 결과 아이템 인벤토리에 추가
- 업적 `ach_craft` 최초 달성 처리
- 3★ 획득 시 업적 `ach_3star` 처리
